# Rock Paper Scissors

---

## 🇬🇧 English

Welcome to the "Rock Paper Scissors" game project brought to life by Onur KOL using Google AI Studio.

This documentation directory gathers the features of the application, its working logic, the technical architecture of the game, and the usage guidelines for integrated external services.

You can access the technical documents and integration guides of the project using the links below:

### 📚 Documentation

- [Game Code Documentation (Architecture and Structure)](./Documentation.md)
  The core file explaining the application's directories, Model-View-Controller (MVC) usage, and module details.

- [Google Play Games & Sign-In Integration](./GPlay_Integration.md)
  A guide explaining how to add Google Sign-In to the application using the `@capawesome/capacitor-google-sign-in` plugin and solutions to potential errors.

- [Start.io (StartApp) Ads Integration](./StartIO_Integration.md)
  A step-by-step guide on displaying Banner and Interstitial ads with Start.io within the application and running them on Capacitor.

### 🎉 Features
- Single Player (Player vs. CPU) and Two Player (Split Screen Offline) modes.
- Google Play Sign-in support.
- Start.io Ads (Full horizontal banner on the main menu, fullscreen ad during screen transitions).
- Advanced Settings (Sound effects, theme/animation management) and Turkish-English Language panel with i18next.
- Persistent Data Storage (localStorage).
- Statistics board (Win / Loss / Draw).

*This application was developed by Onur KOL.*
